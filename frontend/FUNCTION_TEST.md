# 功能测试说明

## 1. 测试环境
- HBuilderX 开发工具
- Android 设备或模拟器

## 2. 测试步骤

### 2.1 录音功能测试
1. 打开应用首页
2. 长按"按住说话"按钮
3. 观察是否显示录音中状态和计时
4. 松开按钮结束录音
5. 观察是否显示录音完成提示

### 2.2 拍照功能测试
1. 点击首页"拍照"按钮
2. 进入拍照页面
3. 点击拍照按钮
4. 观察是否能调用系统相机
5. 拍照完成后是否显示预览
6. 点击"使用照片"按钮确认操作

### 2.3 对话展示功能测试
1. 在首页或聊天页面发送消息
2. 观察消息是否正确显示
3. 检查用户消息和系统消息样式区分
4. 发送多条消息后检查是否自动滚动到底部

### 2.4 页面切换功能测试
1. 点击首页"聊天"按钮
2. 观察是否能正常跳转到聊天页面
3. 在聊天页面点击返回按钮
4. 观察是否能正常返回首页

## 3. 预期结果
- 所有功能正常运行无错误
- 界面显示正常，符合设计要求
- 用户体验流畅，操作响应及时

## 4. 注意事项
- 首次运行需要授权相机和录音权限
- 部分Android设备可能需要手动开启权限
